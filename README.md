# HR Payroll Attendance System

## Overview
An Excel-based HR Management System for attendance tracking, payroll calculation, leave management, employee records, and reporting.

## Features
- Attendance Tracking
- Payroll Calculation
- Leave Management
- Employee Database
- Dashboard Reporting
- Excel Automation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Formulas
- Conditional Formatting
- Data Validation
